# Colecțiile Mele (My Collections)

O aplicație Flutter modernă și responsivă inspirată de Altova My Collections, creată pentru a gestiona inventare personale, hobby-uri și arhive de obiecte.

## Funcționalități principale
- **Autentificare utilizator**: Ecran de login pregătit pentru utilizatorul tău.
- **Panou de control colecții**: Vizualizează toate colecțiile tale într-un format de grilă responsiv, care se adaptează pe mobil, tabletă sau desktop.
- **Detalii colecție**: Explorează lista de elemente dintr-o colecție anume (cu exemple precum Cărți, Filme, Ceasuri, Machete Auto).
- **Design Adaptabil**: Funcționează impecabil pe web, iOS, Android, macOS și Windows.

## Tehnologii folosite
- Flutter & Dart
- Material 3 Design
- Bază de date și stări gestionate local (Mock Data pentru demonstrație)

## Rulare și Instalare
1. Asigură-te că ai [Flutter](https://flutter.dev) instalat.
2. Clonează sau deschide proiectul în editorul tău preferat.
3. Rulează `flutter pub get` pentru a descărca dependențele.
4. Rulează aplicația folosind `flutter run` (poți selecta platforma dorită: Chrome, Edge, emulator Android, etc.).
